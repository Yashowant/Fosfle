1. Basics & Syntax

CSS applies styles to HTML using selectors and declarations. Basic rule: 

selector { property: value; }

External, internal, inline

Prefer external CSS for maintainability.

<link rel="stylesheet" href="styles.css">
<style>
  body { font-family: system-ui; }
</style>
<div style="color:red">Inline</div>

2. Selectors

Common selector types

/* type selector */     p { color: #fff; }
/* class */             .btn { padding: 8px; }
/* id */                #main { max-width: 1200px; }
/* descendant */        .nav a { color: #9aa; }
/* child */             .list > li { margin: 8px 0; }
/* group */             h1, h2, h3 { font-weight:600; }
/* attribute */         input[type="text"] { border-radius:4px; }
/* pseudo-classes */    a:hover, input:focus { outline:none; }
/* pseudo-elements */   p::first-line { font-weight:700; }

Specificity: inline > id > class/attribute/pseudo > element. Later rules override earlier ones when specificity is equal.

3. Box Model

Content + padding + border + margin. Use box-sizing for predictable layouts. 

* { box-sizing: border-box; }
.box {
  width: 200px; padding: 12px; border: 2px solid #123; margin: 16px;
}

4. Display & Position

Control layout with display and position values.

/* display */
.block { display:block; }
.inline { display:inline; }
.inline-block { display:inline-block; }
.none { display:none; }

/* position */
static (default), relative, absolute, fixed, sticky
.relative { position:relative; top:0; left:0; }
.absolute { position:absolute; top:8px; right:8px; }

Sticky requires a scroll container and top value.

5. Flexbox

One-dimensional layout system for rows or columns.

.container { display:flex; gap:8px; align-items:center; justify-content:space-between; }
.item { flex: 1 1 120px; /* grow shrink basis */ }

7. Typography

Fonts, sizes, spacing, and responsive type.

body { font-family: system-ui, -apple-system, "Segoe UI", Roboto, "Helvetica Neue", Arial; line-height:1.45; }
h1 { font-size: clac(1.8rem); }
.small { font-size: 0.85rem; color: var(--muted);} /* use rem for scaling */

Use rem for sizes, font-display for webfonts, and limit line-length for readability (50–75 chars).

8. Colors & Gradients

Color formats and gradients.

/* formats */
color: #0ea5e9;        /* hex */
color: rgba(14,165,233,0.85); /* rgba */
color: hsl(195, 85%, 52%);

/* gradients */
background: linear-gradient(90deg, #06b6d4, #7c3aed);
background: radial-gradient(circle at 20% 20%, #ff7, transparent 30%);

9. Transitions & Animations

Smooth property changes and keyframe animations.

/* transition */
.button { transition: transform 180ms ease, box-shadow 180ms ease; }
.button:hover { transform: translateY(-2px); }

/* keyframes */
@keyframes float {
  0% { transform: translateY(0); }
  50% { transform: translateY(-8px); }
  100% { transform: translateY(0); }
}
.float { animation: float 3s ease-in-out infinite; }

10. Responsive & Media Queries

Adapt layout to viewport size.

/* mobile-first */
.container { padding: 16px; }
@media (min-width: 640px) {
  .container { padding: 24px; }
}
@media (min-width: 1024px) {
  .container { max-width: 1100px; margin:auto; }
}

/* responsive grid */
.grid-resp { grid-template-columns: repeat(auto-fit, minmax(220px, 1fr)); }

11. Advanced (Variables, calc, functions)

Modern CSS features for flexibility.

:root {
  --gap: 12px;
  --brand: #0ea5e9;
}
.container { gap: var(--gap); color: var(--brand); }

/* calc */
.card { width: calc(100% / 3 - 16px); }

/* clamp for fluid typography */
h1 { font-size: clamp(1.2rem, 2.4vw, 2.2rem); }

CSS variables are dynamic and cascade. Use them for theming and responsive tweaks.

12. Tools & Best Practices

  • Normalize/Reset: use a well-tested reset (normalize.css) or minimal reset.
  • Use box-sizing: border-box globally.
  • Prefer logical properties (margin-block, padding-inline) for writing-mode support.
  • Keep specificity low; favor class-based styles over nested selectors.
  • Use DevTools to inspect computed styles, layout, and performance.
  • Consider a utility system (Tailwind) or component CSS with BEM for scale.
