HTML Introduction

HTML stands for HyperText Markup Language. It is used to structure the content on the web.

Example:

<h1>Hello, World!</h1>

This will display the heading "Hello, World!"

HTML Editors

You can write HTML using any text editor such as Notepad (Windows), TextEdit (Mac), or code editors like Sublime Text, or Visual Studio Code. Save your file with a .html extension.

HTML Basic

The basic structure of an HTML document includes the <html>, <head>, and <body> tags.

Example:

<!DOCTYPE html>
<html>
  <head>
    <title>My Page</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
  </body>
</html>

This is the basic structure for an HTML document.

HTML Elements

HTML elements are the building blocks of HTML. For example, <p> defines a paragraph, and <a> defines a link.

Example:

<p>This is a paragraph.</p>

This will display: This is a paragraph.

HTML Forms

Forms allow users to enter data. The <form> element defines an HTML form.

Example:

<form>
  <input type="text" name="name">
  <input type="submit">
</form>

This will create a simple form with a text input and a submit button.

HTML Links

Links are created using the <a> tag. The href attribute specifies the destination URL.

Example:

<a href="https://www.example.com">Visit Example</a>

This will create a link that leads to "https://www.example.com".
